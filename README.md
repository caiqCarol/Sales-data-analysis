# Sales-data-analysis
使用使用 Python 和 SQL 分析了 2022-2023 年的零售数据，识别出畅销产品、区域趋势和销售增长情况，并提供了可操作的见解，以优化物流、提升销售额和提高盈利能力。

项目概述
本项目旨在分析销售数据，以识别影响收入和盈利能力的关键因素。
目标是挖掘有助于提升业务绩效和决策水平的洞察。

项目目标
该项目的主要目标是：
分析销售业绩：找出业绩最佳的产品、地区和类别。
优化物流：了解运输模式及其对销售的影响。
追踪增长情况：比较 2022 年和 2023 年的月度销售增长情况。
找出趋势：确定哪些月份和类别的销售额和利润最高。
提供建议：提供数据驱动的建议，以改善业务运营和盈利能力

数据集
来源：Kaggle 零售订单数据集
关键字段：
Region
Category / Sub-category
Sales (Revenue)
Profit
Quantity

使用的工具
Python：用于数据提取、清洗、转换和加载 (ETL)。使用了 pandas 和 sqlalchemy 等库。SQL：用于高级数据分析，包括聚合、窗口函数和趋势分析。

分析流程
1. 数据清洗
检查缺失值和不一致之处，标准化列名和格式
2. 数据转换
按地区、类别和产品汇总销售额。计算关键指标，例如总收入和利润
3. 探索性数据分析
比较不同产品类别的收入，分析各地区的利润分布，识别销售业绩模式

关键问题
该分析旨在回答以下关键业务问题：
热门产品：收入最高的10种产品是什么？
区域业绩：各区域销量排名前 5 的产品是什么？各地区销量排名第一的类别是什么
销售增长：2022 年和 2023 年的月度销售增长情况如何？
类别表现：每个类别中，哪个月份的销售额最高？
利润增长：与 2022 年相比，2023 年哪个子类别的利润增长最高？

建议
聚焦畅销产品：将更多资源投入到营收排名前十的产品，例如 TEC-PH-10003645 和 FUR-CH-10000454，以实现利润最大化。区域优化：根据区域偏好调整营销策略。例如，TEC-PH-10003645 在南部和西部地区均表现优异。例如，中部东部南部对科技产品更偏好西部对家具产品更偏好。季节性促销：利用月度增长数据洞察，制定季节性促销计划。例如，家具和办公用品分别在三月和五月表现突出。品类拓展：投资于家具和科技等高绩效品类，以推动进一步增长。利润最大化：专注于利润增长最快的子品类，例如科技产品，以确保持续盈利。
